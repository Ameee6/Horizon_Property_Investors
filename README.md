# Horizon Property Investors

![Horizon Property Investors Logo](public/assets/branding/logo.png)

A modern web application for Horizon Property Investors - helping homeowners sell their homes fast and as-is with fair cash offers.

## Branding Assets

The company logo is located at `public/assets/branding/logo.png` and is used throughout the application for:
- Website header and navigation
- Favicon and browser tab icon
- Social media meta tags (Open Graph and Twitter Cards)
- Marketing and documentation materials

### Logo Usage

The logo file is accessible at the path `/assets/branding/logo.png` when the application is running. To use the logo in components:

```tsx
<img src="/assets/branding/logo.png" alt="Horizon Property Investors Logo" />
```

## Development

This project uses Vite + React + TypeScript.

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linter
npm run lint

# Type check
npm run typecheck
```
